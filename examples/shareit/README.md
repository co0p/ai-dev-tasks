shareit
=======

a platform for friends and family to share objects and track the sharing. 

example usecase: I have a cargobike. I want to allow my friends to use it but track who is using it right now. 

solution: the cargobike is added to a sharing catalog and friends can reserve the item. 

Tech: Mobile first website using tailwindcss and some easy frontend framework, golang for the backend, storage is mongodb. local development using docker and deployment to heroku.
